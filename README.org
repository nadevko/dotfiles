My personal [[https://github.com/nix-community/NUR][NUR]]-compatible dotfiles
repository.

** Repository structure

The repository is organized into the following directories:

- hm/ :: Contains the configurations for Home-Manager that I use.
- nixos/ :: Houses the NixOS system configurations.
- droid/ :: Stores nix-on-droid configs, nix-enabled termux fork.
- overlays/ :: Nix overlays.
- lib/ :: Nix helper functions.
- pkgs/ :: Nix packages.
- modules/ :: Nix modules.
  - hardware/ :: Common settings that are useful for specific device models.
  - desktop/ :: Configurations for setting up the desktop environment.
  - users/ :: Modules responsible for managing user configurations.
  - programs/ :: Modules for configuring various applications.

Note: Modules for both home-manager and NixOS are placed together within this
unified structure using ~_class~ option.
