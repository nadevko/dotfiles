** Repository structure

The repository is organized into the following directories:

- [[*Home manager][hm]]/ :: Contains the configurations for home-manager that I
  use.
- [[*NixOS][nixos]]/ :: Houses the NixOS system configurations.
  - [[*kinstra][kinstra]]/ :: Minimal ISO setup for flash drives with repair and
    install tools.
- [[*Droid on nix][droid]]/ :: Stores nix-on-droid configurations, a nix-enabled
  Termux fork.
- [[*Overlays][overlays]]/ :: Nix overlays.
  - [[*maintainer][maintainer]] :: This repository.
  - [[*nur][nur]] :: Packages from nix user repository.
- [[*Libraries][lib]]/ :: Nix helper functions.
  - [[*Trivial][trivial.nix]] :: Trivial and repo-supporting functions.
- [[*Packages][pkgs]]/ :: Nix packages.
- [[*Modules][modules]]/ :: Directories with definitions of option modules.
- [[*Nixpkgs][nixpkgs]]/ :: Provides options to set up channels and overlays.

Note: Modules for both home-manager and NixOS are placed together within this
unified structure using the ~_class~ option.
